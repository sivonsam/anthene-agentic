MetodiPolkuAuthKuvaus
GET /api/health – Health check
GET /api/tools Bearer Kaikki 21 työkalua kategorioineen
GET /api/users/me Bearer Kirjautuneen käyttäjän profiili (luo tarvittaessa)
PATCH /api/users/me Bearer Päivitä oma näyttönimi tai asetukset
GET /api/agents Bearer Omat agentit
POST /api/agents Bearer Luo uusi agentti (201)
GET /api/agents/{id} Bearer Hae yksittäinen agentti
PATCH /api/agents/{id} Bearer Päivitä agentin kenttiä
DELETE /api/agents/{id} Bearer Poista agentti (204)
GET /api/agents/store Bearer Julkiset/jaetut agentit (AgentStore)
POST /api/agents/{id}/copy Bearer Kopioi Store-agentti omaksi
POST /api/run/{agent_id} Bearer Suorita agentti — SSE streaming (token, tool_start, tool_end, done)
POST /api/consult Bearer AI-konsultti — SSE streaming, GPT-4o, ``` recommendation ``` -blokki lopussa
GET /api/admin/users Admin Kaikki rekisteröityneet käyttäjät
PATCH /api/admin/users/{id} Admin Vaihda rooli (admin/editor/viewer) tai aktivoi/estä
DELETE /api/admin/users/{id} Admin Poista käyttäjätietue tietokannasta
GET /api/admin/invites Admin Odottavat kutsut
POST /api/admin/invites Admin Luo kutsu sähköpostilla + roolilla
DELETE /api/admin/invites/{id} Admin Peruuta kutsu
GET /api/agents/admin/all Admin Kaikki agentit (omistajasta riippumatta)
🔑 Bearer = JWT Bearer token (B2C tai dev-token) 🛡️ Admin = role=admin vaatimus ⚡ SSE = text/event-stream, data: {type, content}
